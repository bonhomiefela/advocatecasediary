# advocatecasediary

A minimal Android app (Kotlin) that opens the website https://advocatecasediary.netlify.app/ inside a WebView.

How to run
1. Open this project in Android Studio.
2. Let Android Studio sync (it will generate the Gradle wrapper if missing).
3. Run the app on an emulator or device with internet access.

Notes
- The app requires Internet permission.
- The app enables JavaScript in the WebView to support modern websites.
